# proposals/ — предложения агентов (не применённые)

Сюда агент-куратор (и будущие агенты) пишут предложения по улучшению проекта.
**Это не прод.** Ничего отсюда не применяется автоматически.

Конвенция (GEO-101 + гейт GEO-100):

- Один файл на запуск: `YYYY-MM-DD-<краткий-слаг>.md`.
- Агент только предлагает (находки + рекомендации), правок кода/данных не делает.
- Применение — через ревью человеком/CI и отдельный PR. Агент в `main` не коммитит.

Роль и контракт куратора: см. `agents/curator.md`.
